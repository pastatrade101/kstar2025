# **App Name**: EduPro Hub

## Core Features:

- Course Catalog: A comprehensive listing of all courses offered, with detailed descriptions, schedules, and instructor information.
- Faculty Directory: Searchable directory of faculty members, including contact information, bios, and research interests.
- News & Events: A dynamic feed of the latest news, announcements, and upcoming events at the institution.
- Admissions Information: Detailed information for prospective students, including application procedures, deadlines, and financial aid options.
- AI-Powered Chatbot: An AI chatbot to answer frequently asked questions from prospective and current students.
- Image Generation: AI tool for generating images for any piece of the content on the web page to accelerate content creation and provide visual flair.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Very light blue (#E8EAF6), almost white, creating a clean and professional look.
- Accent color: Analogous purple (#7C4DFF), brighter and more saturated, for interactive elements and calls to action.
- Body font: 'PT Sans', sans-serif, for readability and a modern feel.
- Headline font: 'Playfair', serif, for headings.
- Use clear and consistent icons from a standard library (e.g., Material Design Icons) to represent different sections and actions.
- Maintain a clean and well-organized layout with ample white space to avoid a cluttered appearance.
- Subtle transitions and animations to enhance user experience without being distracting.
